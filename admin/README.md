### 笔记
视觉窗口：根据CSS3规范，视口单位主要包括以下4个：
1. vw：1vw等于视口宽度的1%。
2. vh：1vh等于视口高度的1%。
3. vmin：选取vw和vh中最小的那个。
4. vmax：选取vw和vh中最大的那个。
